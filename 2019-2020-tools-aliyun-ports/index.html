阿里云开放端口教程

, 01/20/2020, Aliyun IP fs

本指南用于在开放阿里云服务器的端口，以供其他用户访问。

因为安全策略，大部分端口默认是无法通过服务器的公网IP进行访问的，本实验课需要使用外网端口，因此需要同学们按照下面的设置打开你的外网端口。

访问地址 https://ecs.console.aliyun.com/ 进入阿里云ECS的控制台。在菜单找到本实例安全组

点击本实例安全组，点击右侧的配置规则

点击本实例安全组，点击右侧的配置规则

在跳转的新页面点击添加安全组规则，在弹框内输入以下配置：

文字版本参考：

规则方向：入网方向
授权策略：允许
协议类型：自定义TCP
端口范围：1/20000
优先级：1
授权类型：IPV4类型地址访问
授权对象：0.0.0.0/0

输入完成后，点击确认即完成设置。
